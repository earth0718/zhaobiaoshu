## 第三章 评审办法


## 第三章 评审办法

### 3.1 总则
1. **评审原则**：遵循公开、公平、公正和诚实信用原则，采用综合评估法，结合定量评分与定性评审。
2. **评审程序**：分资格性审查、符合性审查和详细评审三阶段，任一阶段不通过即终止。
3. **标段独立性**：各标段独立评审，同一投标人可中选多个标段。
4. **评标委员会**：依法组建5人及以上单数评委会，含技术、经济专家及招标人代表。

### 3.2 初步评审（合格性审查）
投标文件须通过以下审查方进入详细评审：

| **审查类别**       | **审查项目**                                                                 | **判定依据**                                       |
|---------------------|-----------------------------------------------------------------------------|--------------------------------------------------|
| **资格性审查**      | 1. 建筑工程施工总承包二级及以上资质证书有效性                                | 资质证书扫描件及“全国建筑市场监管公共服务平台”查验 |
|                     | 2. 近三年（2022-2025）≥2个单项目≥30,000㎡预制构件供货业绩                     | 中标通知书+合同关键页扫描件（含面积、签字盖章）    |
|                     | 3. 有效ISO 9001质量管理体系认证证书                                         | 认证机构官网可查验证书状态                        |
|                     | 4. 无重大质量事故及合同违约的书面声明                                        | 投标人法定代表人签署的声明函                      |
|                     | 5. “信用中国”失信被执行人记录查询结果                                        | 开标当日查询截图（代理机构现场操作）              |
| **符合性审查**      | 1. 投标文件按标段独立封装、签字盖章完整                                       | 封装状态及文件签署规范性                          |
|                     | 2. 实质性响应招标文件技术要求（不可负偏离）                                   | 技术参数响应表（附检测报告证明）                  |
|                     | 3. 电子/纸质投标文件按时提交                                                 | 投标截止时间系统记录及现场签收                    |

### 3.3 详细评审指标体系
通过初步评审的投标文件，按**技术部分（60%）** 和 **商务部分（40%）** 进行综合评分。

#### 3.3.1 技术评分（满分60分）

| **评审子项**         | **评分内容**                                                                 | **分值** | **评分标准**                                                                 |
|----------------------|-----------------------------------------------------------------------------|----------|----------------------------------------------------------------------------|
| **施工组织方案（20分）** | 1. 构件生产计划与施工进度匹配性                                             | 8        | 工期安排合理得6-8分；基本合理得3-5分；存在冲突得0分                            |
|                      | 2. 物流运输方案（含交通管制应对措施）                                       | 5        | 路线规划清晰、时效保障强得4-5分；方案笼统得1-3分；未提供得0分                 |
|                      | 3. **产能保障及应急响应机制（招标文件明确的核心权重项）**                      | 7        | 产能分配明细+备用供应商名单+72小时应急预案得5-7分；仅有简单说明得1-4分         |
| **技术标准符合性（25分）** | 1. 构件技术参数（如混凝土强度、平整度、抗震等级）与招标文件符合度                | 15       | 完全符合得15分；单项参数负偏离每处扣3分（扣完为止）                            |
|                      | 2. 检测能力保障（自有实验室或第三方省级检测协议）                             | 6        | 提供省级检测机构合作协议得6分；地市级机构得3分；无协议得0分                     |
|                      | 3. 不合格品处理流程（退换机制及追溯记录）                                     | 4        | 流程完整可追溯得4分；流程缺失扣2-4分                                         |
| **质量保证体系（15分）** | 1. 质量管控组织架构及关键岗位人员资质                                        | 5        | 配备专职QC工程师（持证）得5分；未明确人员资质扣3-5分                         |
|                      | 2. 原材料（水泥、钢筋）溯源机制                                             | 6        | 提供供应商名录+批次检验报告得6分；缺少溯源文件扣4-6分                           |
|                      | 3. 成品保护方案（运输/现场堆放防损措施）                                      | 4        | 针对性防护方案得4分；通用描述得2分；未提及得0分                               |

#### 3.3.2 商务评分（满分40分）

| **评审子项**         | **评分内容**                                                                 | **分值** | **评分标准**                                                                 |
|----------------------|-----------------------------------------------------------------------------|----------|----------------------------------------------------------------------------|
| **投标报价（30分）** | 1. 报价合理性（以有效报价的算术平均值为基准价）                                  | 30       | 报价等于基准价得满分；每高于1%扣2分；每低于1%扣1分（最多扣10分）              |

|                      | 3. 项目经理经验（近5年同类项目供货管理经历）                                    | 3        | 担任过≥2个项目的得3分；1个项目得1分；无经验得0分                              |

### 3.4 评分计算与定标规则
1. **分值计算**：  
   综合得分 = 技术评分汇总（满分60分） + 商务评分汇总（满分40分）
2. **排名规则**：  
   - 各标段独立按综合得分从高到低排序  
   - 得分相同者，技术得分高者优先；技术得分相同者，报价低者优先
3. **否决条款**：  
   出现以下情形之一作废标处理：  
   - 投标报价高于招标控制价  
   - 技术参数出现实质性负偏离（如混凝土标号低于C50、平整度>3mm）  
   - 未提供180天产能保障方案
4. **定标程序**：  
   评标委员会形成书面报告，推荐1-3名中标候选人（须明确排序），招标人依法确定中标人。

> **附件**：评标委员会技术参数偏离比对表、商务评分计算表、评审意见书（均需评委签字确认）

---

**评审办法设计说明**：  
1. 强化“技术优先”导向，技术部分权重占60%，其中**产能履约能力（7分）** 和**质量追溯机制（6分）** 直接响应招标文件风险管控要求；  
2. 设置阶梯扣分条款（如报价上下浮动扣分、技术参数负偏离单点重扣），避免重大偏离投标人进入候选；  
3. 本地化服务指标（武汉生产基地）提升供应响应效率，降低项目交付风险；  
4. 建立可追溯的审查链条（资质平台验证、检测报告备案、原材料溯源文件），确保评标结果经得起审查。
